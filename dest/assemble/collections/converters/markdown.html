Markdown Helpers: Includes and Blocks

Markdown "Include": Escaped

relative

Derive the relative path from one absolute path to another.

Parameters: string (the value to test against) Default: none

Usage:

[]()

Example:

<a href="dest/assets/css/styles.css"></a>

Renders to: 

<a href="../../assets/css/styles.css"></a>
